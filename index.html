Sharer.js

Create custom social share elements on DOM objects

View on GitHub Download .zip Download .tar.gz

Sharer.js is a very tiny js lib (~1.6kb) to create custom social share components on DOM elements for your website. No dependencies.

Installing:

Using NPM 

npm install sharer.js

Using Bower 

bower install sharerjs --save

Grabbing the latest min version and adding to your page 

<script src="path/to/sharer.min.js"></script>

Or you can use a CDN

<script src="https://cdn.jsdelivr.net/sharer.js/0.2.3/sharer.min.js"></script>

For Meteor Users:

meteor add ellisonleao:sharerjs

Adding share behaviour to a component

Sharer.js currently supports:

  • Twitter
  • Facebook
  • Linkedin
  • Google Plus
  • Email
  • Whatsapp
  • Telegram
  • Viber
  • Pinterest
  • Tumblr
  • Hackernews
  • Reddit
  • VK.com

Twitter:

Available Data attributes:

  • data-title
  • data-url
<button class="sharer button" data-sharer="twitter" data-title="Checkout Sharer.js!" data-url="https://ellisonleao.github.io/sharer.js/">Share on Twitter</button>
