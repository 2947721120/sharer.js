Sharer.js

创建自定义的社交分享的元素在DOM对象

查看 GitHub

Sharer.js 是一个轻量级的js LIB创建的DOM元素为您的网站定制的社会共享组件。无依赖.

Installing:

Using NPM 

npm install sharer.js

Using Bower 

bower install sharer.js

抓住了最新版本的最小和添加到您的网页 

<script src="path/to/sharer.min.js"></script>

或者你可以使用一个CDN

<script src="https://cdn.c2cbc.com/sharer.js/latest/sharer.min.js"></script>

对于用户流星 Meteor :

meteor add ellisonleao:sharerjs

每个组件都有它自己的共享 

data-*
属性.在每个社交媒体网站的基础上，在下面的这些元素.

你也可以通过添加设置弹出窗口大小 

data-width
和 
data-height
属性.

例子:

<button class="sharer button" data-sharer="somesharer" data-width="800" data-height="600" data-title="Checkout Sharer.js!" data-url="https://ellisonleao.github.io/sharer.js/">Share!</button>

共享组件


推特:

可用日期属性:

  • data-title
  • data-url
  • data-via (可选：推特的用户名没有@)
  • data-hashtags (可选：逗号分隔的标签)
<button class="sharer button" data-sharer="twitter" data-title="Checkout Sharer.js!" data-hashtags="awesome, sharer.js" data-url="https://ellisonleao.github.io/sharer.js/">分享 Twitter</button>
